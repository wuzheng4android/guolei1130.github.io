pages搭建个人博客

  • hexo

搭建步骤

  1. 下载nodejs并安装
  2. 安装hexo npm install -g hexo-cli
  1. 初始化gitpage，注意即使用户名大些 也弄小写，然后在电脑中 git clone xxx
  2. 进入本地仓库，初始化hexo站点，hexo init
  3. 修改配置_config.yml，public_dir 为 ./
  4. 修改主题 找到好的主题包，下载zip或者git，将下面主题相关的文件复制到themes/某主题名下
  5. 修改root目录下的_congig.yml文件，修改

deploy:
type: git
repo: https://github.com/Guolei1130/Guolei1130.github.io.git
branch: master

duoshuo:
on: true
domain: guolei1130

是否开启多说评论，http://duoshuo.com/create-site/

使用上面网址登陆你的多说，然后创建站点，在 domain 中填入你设定的域名前半部分

http://<要填的部分>.duoshuo.com (domain只填上<>里的内容，不要填整个网址)

auto_excerpt:
enable: false
length: 150
